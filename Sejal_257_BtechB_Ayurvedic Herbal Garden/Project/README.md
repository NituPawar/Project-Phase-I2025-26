# ?? Ayurvedic Herbal Garden Project

## ?? Project Overview
The **Ayurvedic Herbal Garden Project** is a web-based application designed to provide
information, prediction, and analysis related to medicinal plants used in Ayurveda.
It integrates modern web technologies with Machine Learning to assist users in
understanding herbal growth patterns, medicinal value, and environmental impact.
