# Digital Detox and Mental Well-Being System

## Project Overview
The Digital Detox and Mental Well-Being System is designed to help users
reduce screen addiction and improve focus, emotional balance, and
self-control using detox modes, gamification, and parental supervision.

## Features
- Screen time control and detox mode
- Gamification and reward system
- Parental control with PIN protection
- Emotional awareness and mindfulness support

## Technologies Used
- Flutter
- Firebase
- Python (if backend is used)

## Setup Instructions
1. Install required dependencies
2. Configure Firebase
3. Run the application
4. Login or Register to start using the system

## Usage
- Enable detox mode
- Set screen time limits
- Track progress and achievements
